Beryl Project Branding

In 2023 I was commissioned to create a Logo and branding design for the Beryl project. Beryl is an open source, offline-first, markdown based Todo List system.

For this project, I aimed the capture a clean and consistent feeling, to allow the project itself do the speaking. I was provided with the idea of a barrel in the style of the VLC logo, alongside a beautiful Beryl green colour from the project’s creator. Which from that I created the logo and a complimentary colourscheme that would work in the context of an app or website.
