Useful for making CJK supported fonts for Rockbox, or adding glyphs into a font for web purposes etc. Fontforge is the main application used, however if you intend to make fonts for Rockbox, you’ll need convttf from their dev tools. This guide focuses on combining CJK fonts with non-CJK fonts.
