3. Overlay Images

Â  Drawing overlays in Rockbox is a neat trick to mask sections of the screen to create unique interface designs. While mostly used to create rounded corners on album art, the technique can actually be applied to any part of the screen.

How it works

Â  Rockbox sees a specific RGB Value as transparent (another value it sees as the foreground colour) you can read more about it in the Notes of this section of the Wiki. By putting these into our images, they can have transparent sections. Do however make sure your images are properly formatted to avoid errors, using Gimp this means exporting your images using the 16-bits, R5 G6 B5 option.

Interesting applications

In my own theme, adwaitapod, I used overlays ontop of the main viewport to create rounded corners. Everytime a new item is selected, the overlay refreshes which sometimes has a visible lag before it reappears, however usually this isn’t an issue.

I also used the idea to solve a problem for someone else who wanted to have a squared scrollbar, using one pixel images to mask the corners of the default scrollbar. In situations like these, make sure to use the Cyan colour so it will be drawn as the foreground colour.
