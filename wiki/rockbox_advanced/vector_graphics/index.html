Vector_graphics

In Rockbox themes, there is a missing link between re-colourable graphics and anti-aliased graphics. With all of the classical features, it is impossible to achieve an asset that exhibits both of these properties. However if we take advantage of the font system, we can created accurate vector graphics that behave exactly like images which are anti-aliased and can be re-coloured. This guide will go over my notes and explorations into this new practice.

A Warning

Creating vector graphics in Rockbox is not a perfected art and the process is finicky and time consuming. A detailed process described here is not guaranteed to work for other use-cases, and you will most likely have to try many different things if this happens. Here be dragons.

What you will need

The tools I use are Inkscape, FontForge and the Rockbox font utility ‘convttf’. This process could work using other applications like adobe illustrator, however I have not tested this nor am able to. You will also need access to vector graphics. You can create your own, or find some elsewhere. It is import that these are clean and without errors.
