Background

Â  This site was made to scratch an itch I long had. Sharing stuff on social media has never been my mojo, I’ll often post my photography, however I yearned for something more expansive and exciting so I made a website.

Â  The first version of the site was made by modifying what I felt was a suitable theme, however I soon became aware of my lack of knowledge in web design and development, scrapping the site soon after I finished it. This current version has been built from scratch, using Hugo as a method of generating files offline to upload to github.

Inspiration

Â  Lizbeth Poirier’s beautiful site Ritual Dust was a big inspiration and help when designing the site, Lizbeth has an incredibly helpful collection of resources that I learned a lot from including Fonts, Html and Css. Thomasorus’ Html Tips guide is a must read if your building your own site with minimal knowledge. Without it, I wouldn’t have used Flexboxes or known how to properly organise my code.

Site Philosophy

Â  This site was made with two things in mind: Fun and Accessibility. Each section has it’s own, vibrant theme using Thomasorus’ beautiful css theming solution. The code was also designed with simplicity in mind, there is no javascript used and every element is completely adaptive without the use of media queries. Lastly, all images are completely optimized for fast page loading, thanks to Rekka Bellum’s easy to follow guide! Most pages have 1080p images, but on the large gallery ones, images are reduced to 720p to help with bandwidth.

Fonts

Â  This site uses 3 different fonts. CMRDD Bold is used for links/clickable content. Barlow Black is used for all Headers. Finally, League Spartan is used for the text on the site. Each font is both free and open source, their licenses allowing for web use and modification. Font Squirrel’s Webfont generator was extremely useful in preparing the fonts for the website.

Future Additions

Â  While building the site, decisions were made to make the site more accessible via older browser (using Netsurf as a testing point). However, the theming system doesn’t work and there are some other tweaks I’d like to make to perfect it. On top of that, the accessibility of my website isn’t as good as it could be. Currently all images are captioned and correct etiquette was followed for page layouts, but I’d like to give people options to turn off custom theming and fonts. This however would likely involve adding javascript which I’m slightly reluctant on doing. But we’ll see.
